# Data Cleaning & Reporting Automation

This project focuses on automating data cleaning and reporting workflows using Python and Power BI.  
The goal is to transform raw, inconsistent data into clean, analysis-ready datasets and generate meaningful visual reports with minimal manual effort.

---

## 📌 Project Overview

In real-world datasets, issues like missing values, duplicates, and inconsistent formats slow down analysis.  
This project automates the preprocessing stage and connects the cleaned data to Power BI for interactive reporting and insights.

---

## 🛠 Tools & Technologies Used

- Python (Pandas, NumPy)
- Excel / CSV datasets
- Power BI
- GitHub (version control)

---

## ✨ Key Features

- Automated handling of missing values
- Removal of duplicate records
- Data type correction and standardization
- Cleaned dataset export
- Interactive Power BI dashboard for reporting
- Reduced manual effort and improved reporting efficiency

---

## 🔄 Workflow

1. Load raw dataset using Python
2. Perform data cleaning:
   - Handle missing values
   - Remove duplicates
   - Fix inconsistent entries
3. Export cleaned data
4. Import cleaned data into Power BI
5. Build interactive dashboards and reports

---

## 📊 Power BI Dashboard Includes

- Sales overview
- Monthly trends
- Region-wise performance
- KPI cards for quick insights
- Slicers for dynamic filtering
